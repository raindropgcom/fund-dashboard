## 新增 Gitee Go 国内备胎工作流
- 已创建 `.workflow/gitee-go.yml`：Gitee(国内)抓取+生成 → 推回 GitHub gh-pages 分支。
- 目的：GitHub 境外 runner 抓不到东方财富时的兜底，前端 GitHub Pages 网址不变。
- 需配置：Gitee 仓库(mirror) + Gitee Secret `GH_TOKEN`(GitHub PAT，有 repo 写权限) + Gitee Go 控制台启用定时。
- 注意：Gitee Go 的 runs-on / schedule 语法与 GitHub 略有差异，需在 Gitee Go 控制台确认/选择可用云构建环境。
